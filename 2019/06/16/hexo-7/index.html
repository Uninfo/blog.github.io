    Hexo-填坑

    看那些“大佬”做的迁移教程，身为“网志”小白的我。可谓是的一脸懵，所以打算做期教程。

    拷贝

    #目录
 scaffolds/
 source/
 themes/
 
#文件
_config.yml
 package.json

    注意无需全部拷贝，将一下进行**[目录/文件]拷贝到新建目录**。

    安装

    Windows 本地安装

    • 安装 Git
    • 安装 Node.js

    CentOS 安装

    • 安装 nodejs

    • 安装 npm

      yum -y install nodejs npm --enablerepo=epel

    配置

    #新建目录 Blog，切换目录 Blog。

    #安装 hexo
 npm install -g hexo

#安装模块
 npm install
 npm install hexo-deployer-git --save
 npm install hexo-generator-feed --save
 npm install hexo-generator-sitemap --save

    最后

    生成：

    #配置“用户名称”和“用户邮箱”。
 git config --global user.name "***"
 git config --global user.email "***@qq.com"
 
#生成密钥
 ssh-keygen -t rsa -C "***@qq.com"

    添加：

    #复制 SHH 密钥
- 将 id_rsa.pub 文件内容复制到 Github 。

#添加 SSH 密钥
- 标题：Blog
  - 键：粘贴 id-rsa.pub 的内容
    - 最后确认【添加SSH密钥】

    More info:Links

    部署：

    #博客部署
 hexo clean && hexo g && hexo d
