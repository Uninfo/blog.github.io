“黄叹号”网络分析

2019-07-22- 2021-05-15
411- 1m-
小岛诊所 > Windows篇-网络

有的小伙伴可能会遇见网络异常，突然上不了网了，电脑右下角“网络图标”出现黄叹号。经过自己努力的收集、整理资料。总结了四个常见的问题进行分析，以及解决的方案。

异常①

  • 分析：如果使用静态IP地址，突然出现黄叹号提示，那很可能是你得IP地址被占用一般情况会提示：IP冲突；在IP冲突的时候；
  • 解决：请“打开网络和共享中心–本地连接”修改IP地址即可。

异常②

  • 分析：如果在某种网络环境下，其他人使用正常，而您的网络连接显示为黄叹号，并且IP、DNS都正常，那很可能是计算机的网卡驱动程序出现异常；
  • 解决：此时请下载并重新安装网卡驱动程序，重启计算机后故障即可解决。

异常③

  • 分析：如果在某种网络环境下，其他人使用正常，而您的网络连接显示为黄叹号，并且IP、DNS都正常，那很可能是计算机的网卡驱动程序出现异常；
  • 解决：此时请下载并重新安装网卡驱动程序，重启计算机后故障即可解决。

异常④

  • 分析：如果以上方法都无法解决，而同一网络内的其他用户也显示黄叹号；
  • 解决：那么肯定是接入的路由器、交换机、防火墙等出现故障，请找专业的网络工程师进行解决。

注：以下内容，个人总结；如有不足，还望指出。
