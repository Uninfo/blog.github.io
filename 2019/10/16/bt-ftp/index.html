  1. 公网IP: X.X.X.X（VPS）
  2. 内网IP: X.X.X.X
  3. 映射到公网的域名：xxx.com

安装食用 Frpc

其他版本|Windows 系统

开始食用

解压设置 frpc.ini

[common]
server_addr = 0.0.0.0
server_port = 7000
token = 12345678
tcp_mux = true

[rdp]
type = tcp
local_ip = 127.0.0.1           
local_port = 3389
remote_port = 7001  

[smb]
type = tcp
local_ip = 127.0.0.1
local_port = 445
remote_port = 7002

运行进入frpc.ini文件目录里

Shift + 右击 选择 S 键

.\frpc.exe -c frpc.ini

客户端自启（进阶篇）

同样的道理，我们也需要对客户端设置后台运行和开机自启。借助 winsw 工具可以将frpc注册为windows系统中的服务。

仅下载 WinSW.NET2.exe 或者 WinSW.NET4.exe，它们之间人选其一便可。下载地址

下载 winsw 最新版，为了方便将其重命名为winsw.exe, 将该文件和frpc.exe放在一起，然后新建winsw.xml写入以下内容： 

<service>
    <id>Frp</id>
    <name>Frp</name>
    <description>用 Frpc 发布本地电脑网站到外网/description>
    <executable>frpc</executable>
    <arguments>-c frpc.ini</arguments>
    <logmode>reset</logmode>
</service>

然后使用winsw install和frpc start命令即可将 frpc 安装为系统服务。
win+r 后通过services.msc进入到服务列表页面找到 frp 服务。
