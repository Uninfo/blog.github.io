Croc：一款安全性极强的点对点数据传输工具

2021-08-24- 2021-08-25
554- 2m-
岛主推荐 > 本地P2P传输-本地P2P传输croc

Croc是一款安全性极强的点对点（P2P）数据传输工具，该工具允许两台计算机设备以一种简单和安全的方式来传输文件或文件夹。简而言之，Croc是一款命令行接口工具，在Croc的帮助下，广大研究人员可以通过命令行工具来实现文件传输。

实战演示

工具特性

  • 该工具允许任意两台计算机设备直接进行数据传输；
  • 提供了端到端加密支持（PAKE）；
  • 支持跨平台传输（Windows、Linux和macOS）；
  • 支持多文件传输；
  • 允许断线后的文件续传；
  • 不需要本地服务器或端口转发；
  • 支持IPv6和IPv4；
  • 支持使用Tor之类的代理；

工具安装

安装说明：https://github.com/schollz/croc

下载跳转：https://github.com/schollz/croc/releases)

工具使用

发送文件：多个文件，用文件与文件之间用【空格】隔开

croc fend [file1路径] [file2文件路径] 

croc fend date.text

发送消息：–text “hello world”

croc fend --text  "hello world"

自定义密码：-c “123456”

croc fend -v 123456	date.text	###自定义123456密码发送文件date.text

croc fend -v 123456 --text  "hello world"	###自定义123456密码发送消息

设置中转：croc relay

$ croc relay		###设置中转服务器
[info]  2021/02/16 11:38:59 starting croc relay version v8.6.7-05640cd

$ croc relay --ports 3001		###开启指定服务端口
[info]  2021/02/16 11:39:22 starting croc relay version v8.6.7-05640cd
[info]  2021/02/16 11:39:22 starting TCP server on 3001

$ croc --relay 127.0.0.1:3001 send		###通过服务器中转发送文件

常用组合

croc  --relay 127.0.0.1:3001  send  -c 123456  date.text
###通过IP地址（127.0.0.1:3001）的3001端口，中转文件 date.texe  密码是123456

croc  --relay 127.0.0.1:3001  send  -c 123456  --text  "hello world"
###通过IP地址（127.0.0.1:3001）的3001端口，中转消息 "hello world"  密码是123456
