准备工作

拷贝文件

需要拷贝文件和目录有：

_config.yml
package.json
node_modules
scaffolds
source
themes

配置文件

部署之前需要先修改hexo配置文件
打开_config.yml
找到

deploy:
  type: git
  # repo: https://github.com/用户名/仓库.github.io.git  //将这个注释
  repo: git@github.com:用户名/仓库.github.io.git    //修改为这个
  branch: master

安装软件

安装git、nodejs、build-essential软件

sudo apt-get install -y git && sudo apt-get install nodejs && sudo apt-get install build-essential

查看版本（判断是否安装成功）

nodejs --version && npm --version

配置新秘钥

新秘钥的创建和上传

查看秘钥是否添加（首次选 yes）

ssh -T git@githun.com

创建秘钥

ssh-keygen -t rsa -C "你的邮箱地址"

上传秘钥

cat /home/xxx/.ssh/id_rsa.pub //查看id_rsa.pub

的内容复制到github
点击：头像 -> settings -> SSH and GPG keys

验证sshkey（重新输入）

deepin@deepin-PC:~/Hexo_blog$ ssh -T git@github.com
Hi Uninfo! You've successfully authenticated, but GitHub does not provide shell access. //可以看到我们已经成功了

配置用户名、邮箱

git config --global user.name "用户名"

git config --global user.email "邮箱"

安装HEXO及相关模块

安装hexo、相关模块、 文章部署到 git 的模块

sudo npm install hexo-cli -g && sudo npm install && npm install hexo-deployer-git --save

（下面为选择安装）

sudo npm install hexo-generator-feed --save  // 建立 RSS 订阅
sudo npm install hexo-generator-sitemap --save // 建立站点地图

测试HEXO

hexo clean && hexo g && hexo s  // 本地启动服务
hexo clean && hexo g && hexo d  // 部署网站到设定的仓库
